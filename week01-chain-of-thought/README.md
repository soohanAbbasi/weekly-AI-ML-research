## How to Run the Experiments

Each week has its own folder with a Jupyter notebook. All experiments are designed to run on **free Kaggle GPU** — no expensive hardware needed.

```bash
git clone https://github.com/yourusername/weekly-aiml-research
cd weekly-aiml-research
pip install -r requirements.txt
```

Then open any week's notebook on Kaggle or locally.
